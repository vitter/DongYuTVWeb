## DongYuTVWeb

在电视上观看央视频官方视频，使用 WebView JavaScript 注入的方式操作

## 操作

- 方向上: 上一个频道
- 方向下: 下一个频道
- 左键: 声音减
- 右键: 声音加
- 确定键: 播放/暂停
- 数字键: 换台

> 使用老式的闭路电视操作，适合老人使用！
